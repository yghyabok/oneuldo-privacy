오늘도 - 데이터 삭제 요청

시행일: 2026년 4월 13일

"오늘도" 앱에서 수집된 개인정보의 삭제를 요청할 수 있습니다.

삭제 대상

  • 이메일 주소
  • 프로필 정보 (성별, 출생 년도)
  • 계정 인증 정보

달성 기록 데이터는 개인을 식별할 수 없는 익명 형태로 전환되어 서비스 개선을 위한 통계 목적으로 보관될 수 있습니다.

삭제 요청 방법

방법 1. 앱 내에서 직접 삭제

설정 → 로그아웃하면 기기의 인증 상태가 해제됩니다.

방법 2. 이메일로 삭제 요청

서버에 저장된 모든 데이터의 완전한 삭제를 원하시면 아래 이메일로 요청해 주세요.

  • 받는 사람: freedinoma@gmail.com
  • 제목: [오늘도] 데이터 삭제 요청
  • 본문: 가입 시 사용한 이메일 주소

처리 기간 및 방법

삭제 요청을 받은 후 30일 이내에 다음과 같이 처리합니다.

  • 이메일, 프로필, 계정 정보: 완전 파기
  • 달성 기록: 개인 식별 정보를 제거하여 익명화 처리 (개인정보처리방침 제3조에 따름)

참고

  • 이메일을 등록하지 않은 사용자의 데이터는 기기에만 저장되며, 앱을 삭제하면 함께 삭제됩니다.
  • 삭제 완료 후에도 앱은 정상적으로 사용할 수 있습니다 (로컬 데이터는 유지).
